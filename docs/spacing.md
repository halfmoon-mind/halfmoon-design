# 간격 · 라운딩

## 간격 스케일 (`space.*`, 4px 기반)

| 토큰 | px | 용도 가이드 |
|---|---|---|
| `space.0` | 0 | 리셋 |
| `space.1` | 4 | 아이콘-텍스트 사이 |
| `space.2` | 8 | 밀접한 요소 사이 |
| `space.3` | 12 | 입력 내부 패딩 |
| `space.4` | 16 | 기본 패딩/갭 |
| `space.5` | 20 | — |
| `space.6` | 24 | 카드 내부 패딩 |
| `space.8` | 32 | 섹션 내 블록 사이 |
| `space.10` | 40 | — |
| `space.12` | 48 | 섹션 사이 |
| `space.16` | 64 | 페이지 레벨 여백 |

스케일 밖 값(예: 18px)이 필요해 보이면 대부분 설계 오류다 — 인접 스케일로 스냅한다.

## 라운딩 (`radius.*`)

| 토큰 | px | 용도 |
|---|---|---|
| `radius.sm` | 4 | 체크박스, 태그 |
| `radius.md` | 8 | 버튼, 입력 |
| `radius.lg` | 12 | 카드 |
| `radius.xl` | 16 | 모달, 시트 |
| `radius.full` | 9999 | pill, 아바타 |
